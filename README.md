# New-generic-boundaries-in-the-legume-subtribe-Clitoriinae

Here we provide the alignments and partition files used to assess both the phylogenetic position and the generic limits of the subtribe Clitoriinae.

Clitoriinae_alignment.phy and Clitoriinae_partition_file.nex correspond to the dataset used to evaluate the generic limits within the subtribe.

Clitoriinae_357matK_for_broad_analysis.nex corresponds to the alignment used to assess the phylogenetic position of the subtribe within the NPAAA clade.
